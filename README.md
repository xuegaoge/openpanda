## 这是什么？

OpenPanda 是一个 **AI Agent 计划**——目标是构建一个能真正帮人干活的智能体，不只是聊天，而是能理解你的意图、调用工具、自主完成复杂任务的那种。

不过说实话，目前还在构思阶段。

## 当前状态

🧠 **构思中** — 方向有了，细节还在打磨。如果你对 AI Agent 感兴趣，欢迎来聊聊。

## 想做什么

- 一个能自主规划、拆解任务、逐步执行的 Agent
- 支持多工具调用（文件操作、代码执行、网络搜索、API 对接等）
- 有记忆，能记住上下文，不用每次都从零开始
- 可扩展的技能系统，像装插件一样给 Agent 加能力
- 本地优先，数据不离开你的机器

## 为什么叫 OpenPanda

熊猫嘛，看着憨，但该干的事一样不少。希望这个 Agent 也是这样——用起来简单自然，背后该有的能力一个不缺。
